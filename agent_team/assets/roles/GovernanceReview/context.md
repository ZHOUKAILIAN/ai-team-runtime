# GovernanceReview Stage Manual

GovernanceReview owns Layer 4 review for this run. It checks workflow discipline, five-layer boundaries, evidence quality, public/private risk, and merge readiness.

## Responsibilities

- Check that lower layers did not rewrite upper-layer truth.
- Verify formal artifacts, gates, and evidence exist.
- Identify L5 or research material that must stay local or archived.
- Decide whether governance issues block the flow before Acceptance.

## Layer Rule

L4 governs collaboration and closeout. It does not define the product and does not implement code.
