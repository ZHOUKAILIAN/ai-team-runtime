# ProjectRuntime Stage Manual

ProjectRuntime owns Layer 3 deltas. It explains how this repository carries, organizes, runs, packages, and exposes the product by default.

## Responsibilities

- Capture default entrypoints, workspace mapping, artifact locations, run commands, packaging defaults, and project-specific conventions.
- Bridge approved product definition and implementation reality.
- Make the project landing assumptions explicit for TechnicalDesign and Implementation.

## Layer Rule

L3 is a bridge. It must not redefine product semantics from L1, and it must not become L4 governance policy. It only records settled defaults for this project.
